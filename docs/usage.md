# Getting Started

## First Login

1. Open http://localhost:3000 in your browser
2. Create your admin account (first user is automatically admin)
3. You'll be logged in automatically

## Add Your Library

1. Click **Admin** in the top navigation
2. Go to **Libraries**
3. Click **New Library**
4. Enter:
   - **Name**: Your library name
   - **Path**: Path to your manga folder (e.g., `/manga` or `C:\manga`)
5. Click **Create**

Magi will scan your folder and add all manga automatically.

## Start Reading

- Browse your library on the home page
- Click any manga to start reading
- Use arrow keys or click to navigate pages
- Your progress is saved automatically

## Basic Features

- **Search**: Use the search bar to find manga
- **Collections**: Create custom collections in the Collections tab
- **Users**: Add more users in Admin → Users
- **Settings**: Configure options in Admin → Settings
